# pi-stats

Tracks tool, skill, and extension usage across a pi session.

## Features

- Counts every tool call by tool name
- Tracks skill invocations (via `/skill:name`)
- Tracks extension commands (via `/cmd`)
- `/stats` command to view usage summary
- `stats` tool for the LLM to query its own usage

## Usage

Install and pi auto-discovers it. Run `/stats` to see usage so far.
